## [1.0.0] - 2026-01-11

### Added
- Initial release of Tag for Everything
- Mass tag addition to all notes in a folder
- Support for multiple tags at once
- Untagged notes detection and counting
- Automatic tagging and organization of untagged notes
- Settings interface for configuration
- Support for YAML frontmatter tags
- Preservation of existing tags (no duplicates)
- Automatic creation of target folders
- File name conflict resolution (automatic numbering)
- Desktop and mobile support

### Features
- **Folder Tagging**: Add tags to all existing notes in a specified folder
- **Untagged Management**: Find, tag, and organize notes without tags
- **Flexible Tag Input**: Accept tags with or without # symbol
- **Smart Frontmatter**: Create or update YAML frontmatter as needed
- **Batch Processing**: Handle large numbers of files efficiently
- **Real-time Count**: Display number of untagged notes
- **Safe Operations**: Preserve existing tags and content
